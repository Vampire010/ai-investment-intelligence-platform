Live App Deployment

This GitHub Pages site is the public project landing page. The full interactive app requires a Python backend, so deploy it with Render, Railway, Fly.io, or another Docker-capable host using the included deployment files.

  • render.yaml for Render Blueprint deployment
  • railway.json for Railway Docker deployment
  • fly.toml and Dockerfile for Fly.io
  • Health endpoint: /api/health
